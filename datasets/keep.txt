# 单词学习 APP 产品需求文档（PRD）
**文档版本**：V1.0
**编写角色**：需求分析师
**适用版本**：V1.0 基础版

---

## 一、项目概述
### 1.1 需求背景
当前英语学习者普遍存在**记不住、不会用、难坚持**三大痛点，传统背词工具记忆逻辑单一、场景化弱、激励不足。本产品旨在通过**科学记忆算法 + 场景化学习 + 长效激励**，打造高效、易用、可坚持的单词学习工具。

### 1.2 产品定位
以**艾宾浩斯记忆算法**为核心，面向学生、职场人、出国备考用户，提供**词书管理、智能背词、复习巩固、激励成长、辅助工具**一体化单词学习 APP。

### 1.3 核心目标
- 完成完整学习闭环：选词书 → 制定计划 → 学习 → 复习 → 测试 → 巩固
- 提升用户坚持率：每日任务完成率 ≥ 60%，连续 7 日打卡率 ≥ 30%
- 支持多场景：应试、日常、职场、出国英语全覆盖
- 支持离线学习、多端同步、数据安全

### 1.4 目标用户
- 核心：大学生（四六级、考研）
- 次要：雅思/托福/GRE 考生、职场英语用户
- 潜在：K12 学生（需青少年模式）

## 三、功能模块需求（按模块拆分）

本模块为 APP 基础底座，负责用户账号管理、数据同步、权限控制，保障用户学习数据的安全与连续性。
1.1 注册与登录
优先级：P0功能描述：为用户提供账号注册、登录、账号找回功能，降低准入门槛，保障学习数据云端同步。核心业务规则：
支持登录方式：手机号验证码登录、微信授权登录、Apple ID 登录（iOS 端）；
首次登录用户，必须完成新手引导：选择学习目标、对应考试 / 场景、预计考试时间、每日背词量，系统自动生成初始学习计划；
同一账号最多同时登录 2 台设备，超出后最早登录的设备自动下线，学习数据实时云端同步；
未登录用户可使用游客模式：限免体验部分基础词书，学习数据仅本地存储，登录后可选择同步至账号，游客模式不可使用社区、云同步、付费功能。
交互逻辑：APP 启动默认进入登录页，提供登录入口与游客模式入口；登录成功后，首次登录进入新手引导，非首次登录直接进入首页学习 Tab。
功能目标：降低用户注册门槛，保障数据安全，快速完成用户初始学习配置。
1.2 个人信息与学习档案
优先级：P0功能描述：支持用户管理个人基础信息，可视化展示全周期学习数据，让用户清晰感知学习成果。核心业务规则：
个人信息：支持修改头像、昵称、学习目标、个性签名，展示累计学习天数、背词总量、连续打卡天数；
学习档案：可视化展示每日背词数据、词汇量增长曲线、记忆留存率、学习时长分布、测试正确率等核心数据，支持按日 / 周 / 月维度查看。
功能目标：通过数据可视化，给予用户正向反馈，强化学习动力。
1.3 青少年模式
优先级：P1功能描述：符合未成年人保护相关法规，为青少年用户提供合规、健康的学习环境。核心业务规则：
青少年模式需监护人设置独立密码，开启后限制每日使用时长（默认每日 22:00 - 次日 6:00 禁用，单日累计使用不超过 40 分钟）；
仅开放课内同步词书、基础学习功能，关闭社区、付费、陌生人互动等功能；
屏蔽广告、营销类内容，仅展示适龄学习内容。
功能目标：满足合规要求，覆盖 K12 潜在用户群体。
模块二：词书管理模块（P0）
本模块是背词功能的核心基础，负责全场景词书供给、订阅管理、自定义词书能力，满足用户不同的学习需求。
2.1 官方词书库
优先级：P0功能描述：为用户提供全品类、合规的官方词书，支持分类筛选、搜索、预览，快速匹配用户学习目标。核心业务规则：
词书分类维度：应试类（四六级、考研、雅思、托福、中高考等）、场景类（商务英语、旅游英语、影视台词等）、基础类（课内同步、新概念、词根词缀等）；
词书详情页展示：词汇总量、适用人群、词书简介、大纲、用户评分、学习人数，支持免费预览前 20 个单词；
词书分为免费词书与会员专属词书，免费词书全量开放，会员词书仅开通会员的用户可订阅使用；
支持词书收藏、关键词搜索，支持按热度、更新时间、词汇量排序。
交互逻辑：进入词书 Tab，默认展示分类导航 + 推荐词书，点击分类切换词书列表，点击词书进入详情页，可选择订阅 / 解锁。
功能目标：覆盖用户全场景词书需求，让用户快速找到适配的学习内容。
2.2 词书订阅与切换
优先级：P0功能描述：支持用户订阅多本词书，灵活切换当前学习的主词书，管理个人学习词库。核心业务规则：
用户可同时订阅多本词书，但仅可设置一本为「当前学习词书」，纳入每日学习计划；
切换主词书后，系统会基于新词书与用户设置，重新生成学习计划；
已订阅词书支持取消订阅，取消后不清除用户已学习的单词数据。
功能目标：满足用户多目标、多阶段的学习需求，灵活管理学习内容。
2.3 自定义词书
优先级：P1功能描述：支持用户自主创建、导入个性化词书，满足非标、定制化的学习需求。核心业务规则：
支持 3 种创建方式：手动录入单词、批量导入（Excel 模板、文本粘贴）、拍照识别导入；
自定义词书支持编辑单词释义、例句、个人备注，支持修改词书名称、封面；
自定义词书完全纳入学习计划体系，享受和官方词书一致的学习、复习、算法适配能力；
权限限制：普通用户最多创建 5 本自定义词书，单本上限 2000 词；会员用户无创建数量限制，单本上限 10000 词。
交互逻辑：词书 Tab - 我的词书 - 新建自定义词书，选择创建方式，按引导完成创建，成功后可设置为学习词书。
功能目标：满足用户个性化学习需求，提升产品适配性。
2.4 生词本 / 熟词本管理
优先级：P0功能描述：为用户提供单词个性化分类管理能力，强化薄弱单词记忆，过滤已掌握单词。核心业务规则：
生词本：用户在学习 / 查词过程中，可一键将单词加入生词本，支持按词书、添加时间筛选，支持批量加入复习计划；
熟词本：用户可将已完全掌握的单词标记为熟词，标记后该单词不再出现在学习、复习计划中，支持取消标记恢复学习。
功能目标：让用户的学习更有针对性，提升背词效率。
模块三：核心背词学习模块（P0）
本模块是 APP 的核心功能，定义用户单词学习的全流程、记忆规则、内容展示逻辑，决定核心学习体验与记忆效果。
3.1 个性化学习计划生成与调整
优先级：P0功能描述：基于用户的学习目标、考试时间、每日背词量，自动生成「新学 + 复习」的每日学习计划，支持灵活调整。核心业务规则：
计划生成逻辑：用户设定目标词书、考试时间、每日新学单词量，系统基于艾宾浩斯记忆曲线，自动计算每日需复习的单词量，生成当日学习任务；
计划调整：支持用户随时修改每日新学单词量、目标词书、考试时间，调整后系统自动重新计算后续学习计划；
任务优先级：默认复习任务＞新学任务，用户需先完成当日复习，再进行新单词学习（可在设置中关闭该规则）；
补卡规则：用户漏学的任务，可通过补卡完成，单日最多补卡 3 天；普通用户每月 2 次补卡机会，会员用户每月 10 次补卡机会。
交互逻辑：首页核心区域展示今日新学 / 复习任务进度条，点击「设置」可调整计划参数，完成任务后自动标记今日学习完成。
功能目标：为用户制定科学的学习计划，降低决策成本，保障记忆效果。
3.2 单词学习核心流程与记忆分级
优先级：P0功能描述：定义新单词学习的核心交互流程，基于用户反馈动态划分单词掌握等级，匹配对应的复习周期。核心业务规则：
单词掌握度分为 5 个等级，对应不同复习间隔：陌生（0 级，当日重复）、模糊（1 级，1 天后复习）、认识（2 级，3 天后复习）、熟悉（3 级，7 天后复习）、掌握（4 级，15 天后复习）；
单单词学习流程：展示单词 + 英 / 美发音→用户自主判断「认识 / 模糊 / 不认识」→展示单词全维度内容→完成当前单词学习，进入下一个单词；
记忆动态调整：用户选择「不认识」，单词保持 0 级，当日重复出现 2 次；选择「模糊 / 认识」，对应升级等级，按规则进入复习周期；学习过程中支持一键加入生词本、标记熟词、添加个人笔记。
交互逻辑：点击首页「开始学习」进入学习页，左右滑动切换单词，点击对应掌握度按钮完成当前单词学习；页面顶部展示学习进度，完成全部任务后弹出学习完成弹窗，展示本次学习数据与打卡入口。
功能目标：建立标准化的学习流程，基于用户个体记忆数据动态适配，最大化记忆留存率。
3.3 多维度单词内容展示
优先级：P0功能描述：为每个单词提供全维度学习内容，帮助用户理解、记忆、应用单词，拒绝机械记拼写。核心业务规则：
单词标配内容：英 / 美音标、真人发音、核心释义、词性变化、场景化 / 真题例句、常用搭配、词根词缀、同义词 / 反义词；
应试类词书额外配套：单词考点、真题考频、历年真题例句；
交互能力：支持发音语速调节、例句跟读、发音测评，支持内容收起 / 展开，默认展示核心释义。
功能目标：让用户不仅记住单词，更能理解用法、适配考试 / 场景需求。
3.4 多模式背词切换
优先级：P1功能描述：提供多种背词模式，适配不同用户的学习习惯与场景，多维度强化记忆效果。核心业务规则：
标配背词模式：看词辨义（默认）、听音辨词、拼写默写、汉译英；
用户可在学习前 / 学习中自由切换模式，不同模式对应不同交互形式；
拼写模式下，支持键盘 / 手写输入，输入错误时提示正确拼写，错误单词自动加入错题本。
功能目标：适配不同学习场景，强化多感官记忆，提升记忆效果。
模块四：智能复习巩固模块（P0）
本模块是解决 “背了就忘” 的核心，基于科学记忆算法实现智能复习，配套错题、测试功能，帮助用户巩固学习成果。
4.1 智能复习任务
优先级：P0功能描述：基于艾宾浩斯遗忘曲线与用户个体记忆数据，自动生成每日复习任务，强化单词记忆留存。核心业务规则：
复习触发规则：根据单词的掌握等级、上次学习时间，自动计算最佳复习时间，到期自动纳入当日复习任务；
复习记忆调整：复习流程与学习流程一致，用户再次判断掌握度，答对升级、答错降级，降级单词缩短复习间隔，高频重复出现；
支持「快速复习」模式，精简单词展示内容，提升复习效率；
支持用户手动添加单词到复习列表，临时调整复习时间。
功能目标：用科学算法对抗遗忘，最大化单词长期留存率。
4.2 错题本管理
优先级：P0功能描述：自动收录用户学习、复习、练习过程中答错的单词，提供专项复习能力。核心业务规则：
自动收录规则：拼写错误、词义判断错误、听音辨词错误的单词，自动加入错题本，记录错误次数、错误类型；
支持按词书、错误时间、错误次数、掌握程度筛选错题；
支持错题批量移除、标记掌握、加入复习计划、导出；
系统自动标记高频错题，优先推送复习。
功能目标：帮助用户定位薄弱环节，针对性查漏补缺。
4.3 学习测试功能
优先级：P1功能描述：提供多类型测试功能，帮助用户检验学习效果，定位知识漏洞。核心业务规则：
测试类型：单元测试（对应词书的单元词汇）、随机测试（自定义词量 / 范围）、考前模考（对应考试的高频词汇）；
测试题型：选词填空、看词辨义、听音辨词、拼写题，覆盖核心记忆维度；
测试完成后生成详细报告，展示正确率、错误单词清单、薄弱点分析，错误单词自动加入错题本，报告支持保存、分享。
功能目标：帮助用户检验学习成果，强化学习动力，适配应试用户的模考需求。
模块五：激励与成长体系模块（P1）
本模块核心解决用户 “难以坚持” 的痛点，通过多维度正向激励，帮助用户养成长期学习习惯。
5.1 学习打卡体系
优先级：P0功能描述：用户完成当日学习任务后，可完成打卡，记录学习连续性，给予正向反馈。核心业务规则：
打卡触发条件：完成当日全部新学 + 复习任务，即可完成当日打卡，获得成长值、徽章碎片奖励；
展示连续打卡天数、累计打卡天数，通过打卡日历可视化展示用户全周期学习记录；
打卡完成后，支持生成个性化打卡海报，分享至社交平台。
功能目标：强化用户学习仪式感，培养每日学习习惯，通过分享带来社交裂变。
5.2 成就与徽章体系
优先级：P1功能描述：设置多维度成就任务与专属徽章，给予用户阶段性正向激励，提升学习动力。核心业务规则：
徽章分类：学习时长类、连续打卡类、词汇量突破类、测试类、特殊纪念类；
用户完成对应成就条件，自动解锁徽章，徽章展示在个人主页，形成学习身份标识；
稀有徽章解锁后，可获得额外成长值、会员体验卡等奖励。
功能目标：通过阶段性目标与荣誉激励，提升用户长期学习的驱动力。
5.3 组队打卡功能
优先级：P2功能描述：支持用户创建 / 加入打卡小队，通过同伴监督、团队激励，提升学习坚持率。核心业务规则：
小队规则：创建者设置打卡目标、小队人数上限、打卡周期，用户可通过邀请码加入小队；
小队内展示成员每日打卡情况、学习数据，设置小队排行榜，未打卡用户可被提醒；
完成周期打卡目标的小队，全员可获得对应奖励。
功能目标：通过社交监督提升用户留存，解决独自学习难以坚持的痛点。
模块六：辅助工具模块（P1）
本模块为核心学习流程提供配套能力，拓展产品使用场景，提升用户体验。
6.1 拍照查词
优先级：P1功能描述：支持用户拍照 / 导入图片识别单词，快速查询释义、加入生词本，适配阅读场景的即时查词需求。核心业务规则：
支持拍照实时识别、相册图片导入识别，支持整段文本识别，点击单词即可查看详情；
查词记录自动保存，支持批量加入生词本 / 自定义词书；
支持离线查词（需提前下载离线词库）。
功能目标：拓展产品使用场景，满足用户阅读、做题时的即时查词需求。
6.2 单词发音测评
优先级：P1功能描述：基于语音识别技术，对用户的单词 / 例句跟读发音进行评分，纠正发音问题。核心业务规则：
支持单词、例句跟读，从音标、重音、流利度三个维度给出评分与纠错建议；
发音记录自动保存，用户可回放自己的发音与标准发音对比；
多次测评不通过的单词，自动标记加入生词本。
功能目标：解决用户 “只会认不会读” 的痛点，完善单词学习的全维度能力。
6.3 学习提醒
优先级：P1功能描述：提供多维度的学习提醒功能，帮助用户养成固定学习习惯，避免遗忘学习任务。核心业务规则：
支持用户自定义提醒时间、提醒频率、提醒内容；
提醒类型：每日学习提醒、复习提醒、打卡提醒、活动提醒；
支持免打扰时段设置，可在系统设置中关闭对应提醒。
功能目标：提升用户打开频次，保障学习计划的连续性。
6.4 离线学习
优先级：P1功能描述：支持用户提前下载词书内容、发音资源，无网络环境下也可正常完成学习、复习任务。核心业务规则：
用户可选择下载对应词书的离线资源，包括单词释义、发音、例句；
离线模式下的学习数据，会在网络恢复后自动同步至云端；
支持设置仅 WiFi 环境下下载资源，避免消耗用户移动流量。
功能目标：适配通勤、无网络等场景，保障学习不中断。
模块七：社区互动模块（P2）
本模块为 V1.0 版本的拓展功能，核心提升用户粘性与产品活跃度，搭建学习交流生态。核心功能点：
学习动态发布：支持用户发布学习打卡、心得笔记、背词技巧等内容，支持配图、话题标签；
内容广场：按算法推荐优质学习内容，支持点赞、评论、收藏、转发，关注用户可在关注页查看专属动态；
话题专区：设置官方热门话题（如 #考研单词打卡 #、# 四六级冲刺 #），引导用户参与内容创作与交流；
内容审核：所有用户发布内容需经过机审 + 人审，过滤违规、侵权、广告内容，保障社区内容生态合规。
